About me

　　我是楊粒平，自出生便在臺中居住成長至今。我畢業於朝陽科技大學資訊與通訊系，並在在學期間對寫程式摸索出一些興趣。大學畢業後，我考上了國立彰化師範大學資工系碩士班，主要做有關無限通訊系統方面的研究。碩士班畢業後，我前往僑光科技大學擔任研究計畫專任助理，負責協助系內事務，並在一年後因聘期以至的緣故離職。 離職之後我深刻感覺到自己的專業能力還不足以找到跟需要專業技術的工作，因而開始投入前端領域的學習。我從2020年12月7日開始接受職訓局的電子商務網站建置班的課程培訓，職訓期間我秉持著不能夠什麼都要老師教到才開始學的學習態度，努力培育自己的前端開發能力。 在學期間，我留意到我學習到的技術如果不去用的話就很容易忘記的問題，因此我不時就會去想以後開發網頁可能會用到的功能並進行實作，藉以輔助記憶與磨練自己的能力。我認為自己還作品集強化自己，因此在3月的時候運用空閒時間著手撰寫介紹自己的單頁式網頁，並於4月中旬的時候完成初步的版面設計。 2021年3月，我開始進行畢業專題的開發。在這個專題中，由於其他的組員對後端較不熟悉的緣故，因此我主要負責後端 PHP 程式的撰寫與前後端的 API ，並使用 Vue 寫了購物車資料異動、會員資料更改的功能。同時我也不時地匯去指導組員撰寫網頁的排版，自己也寫了幾個頁面的排版，諸如產品列表、關於我們…等頁面。在這個專題中，我認為自己前端開發的能力有所進步，同時對後端也有了一些基本的認知。我期許能夠將在這邊所培訓的能力投入到職場之中。 感謝您撥冗閱覽我的自傳，我期盼能有機會加入貴公司共同開發網頁。
